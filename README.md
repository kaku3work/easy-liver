# ライブ配信の光と影

一人のリスナーが経験した出来事を通じて、ライブ配信における「応援」「信頼」「倫理」を考えるWebノンフィクション作品です。

## 公開方法（GitHub Pages）

1. このフォルダの中身をGitHubリポジトリへアップロードします。
2. GitHubの **Settings → Pages** を開きます。
3. **Build and deployment** の Source を **Deploy from a branch** に設定します。
4. Branch に `main`、Folder に `/ (root)` を指定して保存します。
5. 数分後、表示されたURLから公開ページを確認します。

## ファイル構成

